# dBot

## How to use setup_py script
Use the script to install dependencies locally and provision and activate the local venv.
1.  Run 
        ```
        chmod +x setup_py.sh
        ```
    to make the script executable
2. Run the script using: 
        ```
        ./setup_py.sh
        ```
